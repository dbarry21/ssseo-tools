Smart SEO automation tools for developers and agencies. Includes schema markup, OpenAI integration for meta descriptions and titles, bulk SEO updates, and more.

== Description ==

**SSSEO Tools** is an all-in-one SEO optimization plugin designed for professionals. Whether you're generating schema for local businesses, automating meta content with AI, or pulling YouTube videos into your site, this modular toolkit helps manage it all from your WordPress dashboard.

=== Features ===

* Generate Local Business, Organization, and FAQ Schema (JSON-LD)
* Create AI-generated meta descriptions and SEO titles with OpenAI
* Embed latest videos from your YouTube channel
* Perform bulk Yoast SEO operations across content types
* Enable/disable Services, Products, and other custom post types
* Built-in admin interface with Bootstrap layout

=== Installation ===

1. Upload the plugin folder `ssseo-tools` to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to `Settings > SSSEO Tools` to configure options.

=== Frequently Asked Questions ===

= How do I use the AI features? =  
Enter your OpenAI API key in the AI tab and click 'Generate' on any supported content type.

= Does this plugin work with Yoast SEO? =  
Yes, bulk tools are designed to work alongside Yoast settings and modify meta robots tags directly.

= Can I disable unused modules? =  
Yes. Use the "Plugin Modules" settings tab to toggle individual components.

= Will you add support for more schema types? =  
Yes. Support for Review, Event, and Service schemas is planned.
